# Project Setup Guide

Esta pasta existe para **recriar o esqueleto original** do projeto. Ela tem
valor histórico e de bootstrap, mas não é a fonte principal de verdade do
estado atual da aplicação.

## Quando usar

- para entender como o esqueleto inicial foi gerado
- para recriar uma base mínima em ambiente separado
- para consultar a estrutura inicial do projeto

## Quando não usar

- como referência principal de arquitetura atual
- como documentação do alpha público
- como fonte de contratos atualizados

## Comando

```bash
cd project_setup
chmod +x run_all.sh
./run_all.sh
```

## Observação importante

O projeto real já evoluiu além do que estes scripts geram. O estado atual está
documentado no `README.md` da raiz e em `docs/planning/`.
