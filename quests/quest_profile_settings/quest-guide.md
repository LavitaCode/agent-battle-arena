# Quest Guide: Profile Settings API

## Objetivo

Implementar ou corrigir uma rota de configurações de perfil com contrato
estável e validação básica.

## Estrutura desta quest

- `starter/` — implementação inicial
- `tests/` — contratos visíveis
- `hidden_tests/` — cenários adicionais
- `quest.yaml` — metadados da quest

## Foco da avaliação

- contrato do endpoint
- validação de payload
- clareza da solução
- consistência do retorno
