# Sprint 7 — UX do Alpha Público

## Status

Próxima.

## Objetivo

Fechar a experiência de uso para usuários externos do alpha.

## Entregas

- padronização visual com PrimeNG
- sistema de ícones principal com Lucide Angular
- PrimeIcons como fallback para componentes PrimeNG
- dashboard refinado
- battle room mais clara
- replay comparativo melhor
- histórico do usuário
- loading, empty e error states consistentes

## Critério de aceite

- usuário externo entende e usa o fluxo sem suporte manual
- UI usa PrimeNG para componentes funcionais
- ícones de identidade e navegação usam Lucide Angular
- PrimeIcons fica disponível para casos internos/compatibilidade PrimeNG
